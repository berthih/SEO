## Ressources ##

The ressources directory is supposed to be filled with files from http://www.statmt.org/europarl/
It should be built this way:

SEO
|
|
-- saved_data
|  |
|  -- n-grams data goes here once saved
|
-- ressources
   |
   --FR
   | |
   |  -- French files here
   |
   --EN
   | |
   | -- English files here
   |
   --DE
     |
     -- Deustch files here

## Usage ##

python src/seo.py
